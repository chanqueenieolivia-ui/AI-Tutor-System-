# AI Tutor System 🎭

一個沉浸式 AI 導師系統，結合 **音、影、表情、回覆、錯誤回放、分數計算、學習曲線追蹤、個人化課程、互動劇情分支、跨文化融合、觀眾參與** 等模組，打造教育 + 娛樂 + 文化的互動舞台。

---

## 🚀 功能模組
- 🎵 **[LipSync 嘴型同步](ca://s?q=LipSync_嘴型同步模組)**：語音與嘴型精準對齊  
- 🎥 **[場景切換平滑化](ca://s?q=場景切換平滑化模組)**：淡入淡出過渡動畫  
- 😀 **[多模態回覆](ca://s?q=多模態回覆模組)**：語音 + 文字 + 表情同步  
- 🔄 **[錯誤回放](ca://s?q=錯誤回放模組)**：學員表演與正確版本對比  
- 📊 **[分數計算](ca://s?q=分數計算模組)**：多維度評分與即時評語  
- 📈 **[學習曲線追蹤](ca://s?q=學習曲線追蹤模組)**：生成進步趨勢圖表  
- 🎯 **[個人化課程](ca://s?q=個人化課程模組)**：根據弱點自動推薦課程  
- 🎭 **[互動劇情分支](ca://s?q=互動劇情分支模組)**：表現不同 → 劇情走向改變  
- 🌏 **[跨文化融合](ca://s?q=跨文化融合模組)**：粵劇、西方音樂劇、舞蹈、語言、遊戲文化  
- 👥 **[觀眾參與](ca://s?q=觀眾參與模組)**：即時投票影響劇情
